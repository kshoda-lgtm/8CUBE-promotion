### 🔧 **使用手順（v4.0）**

#### **Step 1: 環境準備**
```bash
# 必要なライブラリをインストール
pip install python-pptx google-generativeai
```

#### **Step 2: Gemini APIキー取得**
1. Google AI Studio にアクセス: https://makersuite.google.com/app/apikey
2. 「Create API Key」でAPIキーを生成
3. 環境変数に設定：
```bash
# Windows
set GEMINI_API_KEY=your_api_key_here

# または、スクリプト内に直接記載（テスト用）
```

#### **Step 3: PowerPoint処理（単一ファイル）**
```bash
cd "promotion部"
python powerpoint_processor_gemini.py
# PowerPointファイルパスを入力
```

#### **Step 4: PowerPoint処理（バッチ処理）**
```bash
cd "promotion部"
python batch_process_gemini.py "AIマニュアル化\AIマニュアル化"
# フォルダ内の全.pptxファイルを自動処理
```

#### **Step 5: JSON処理**
1. 生成されたJSONをGoogle Driveにアップロード
2. Google Apps Scriptで `processJSONFiles()` 実行
3. スプレッドシートに各項目が正確に振り分けられて登録完了
