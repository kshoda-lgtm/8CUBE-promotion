## 🛠️ 技術仕様（無料版）

### 🔄 **新アプローチ：Python事前処理方式**

**完全無料でPowerPoint自動処理を実現！**

#### **システム構成**

```
PowerPoint(.pptx) → [Python + python-pptx] → JSON → [Google Apps Script] → Spreadsheet
```

#### **必要なサービス・ツール**
- ✅ **Python 3.x**（無料）
- ✅ **python-pptx ライブラリ**（無料）
- ✅ **Google Workspace**（既存）
- ✅ **Google Apps Script**（無料）
- ✅ **Google Drive**（無料枠）

#### **技術的利点**
- 🚀 **ファイルサイズ制限なし**（ローカル処理）
- ⚡ **Google Apps Scriptの6分制限回避**
- 🎯 **高精度抽出**（python-pptx使用）
- 💰 **完全無料**（追加費用0円）

#### **処理能力**
- **ファイルサイズ**: 制限なし（100MB以上でも対応）
- **同時処理**: ローカル環境次第
- **抽出精度**: 90%以上（テキスト要素）
- **処理時間**: 1ファイル30秒～2分

### 📁 **実装ファイル**

| ファイル名 | 役割 | 実行環境 |
|-----------|------|----------|
| `powerpoint_processor.py` | PowerPoint解析・JSON変換 | ローカル |
| `JSON_processor.js` | JSON取り込み・DB登録 | Google Apps Script |
| `Code.js` | 従来の処理（バックアップ） | Google Apps Script |

### 🔧 **使用手順**

#### **Step 1: Python環境準備**
```bash
pip install python-pptx
```

#### **Step 2: PowerPoint処理**
```bash
cd "promotion部"
python powerpoint_processor.py
# PowerPointファイルパスを入力
```

#### **Step 3: JSON処理**
1. 生成されたJSONをGoogle Driveにアップロード
2. Google Apps Scriptで `processJSONFiles()` 実行
3. スプレッドシートに自動登録完了
